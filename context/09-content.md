## Lesson Format

Moi bai hoc trong database gom:

```typescript
{
  title: "Bien trong JavaScript",
  slug: "bien-trong-javascript",
  summary: "Tom tat ngan gon noi dung bai hoc (AI generated)",
  content: "Noi dung chi tiet hon (optional, co the null)",
  externalLinks: [
    { title: "MDN - Variables", url: "https://...", type: "documentation" },
    { title: "Video bai giang", url: "https://...", type: "video" }
  ],
  estimatedMins: 30,
  trackOrder: 5
}
```

### Tren giao dien hien thi:
1. **Summary** - tom tat noi dung bai hoc
2. **AI Chat box** - hoi ve noi dung bai, AI biet context bai nay
3. **External Links** - link tai lieu tham khao
4. **Quiz button** - lam bai kiem tra

## Quiz Types

### 1. Single Choice (Trac nghiem)
```json
{
  "questionType": "SINGLE_CHOICE",
  "questionText": "Keyword nao dung de khai bao bien trong JS?",
  "options": [
    { "id": "a", "text": "var" },
    { "id": "b", "text": "let" },
    { "id": "c", "text": "const" },
    { "id": "d", "text": "Tat ca cac dap an tren" }
  ],
  "correctAnswer": ["d"],
  "explanation": "var, let, va const deu la keyword de khai bao bien..."
}
```

### 2. Essay (Tu luan - AI cham)
```json
{
  "questionType": "ESSAY",
  "questionText": "Giai thich su khac biet giua let va const. Khi nao nen dung cai nao?",
  "options": null,
  "correctAnswer": null,
  "explanation": "Dap an mau de AI tham khao khi cham diem..."
}
```
- User viet cau tra loi
- Backend gui cau tra loi + dap an mau den AI
- AI cham diem (0-100) va giai thich

### 3. Code Challenge (Viet code)
```json
{
  "questionType": "CODE_CHALLENGE",
  "questionText": "Viet ham tinh tong cac so trong mang",
  "codeTemplate": "function sumArray(arr) {\n  // Viet code cua ban o day\n}",
  "testCases": [
    { "input": "[1, 2, 3]", "expected": "6" },
    { "input": "[10, 20, 30]", "expected": "60" },
    { "input": "[]", "expected": "0" }
  ],
  "explanation": "Co the dung reduce() hoac vong lap..."
}
```
- User viet code trong Monaco Editor
- Backend chay code voi test cases
- Tra ve ket qua pass/fail cho tung test case

## Content Creation Strategy

1. **AI generate** lesson summaries va quiz questions
2. **Ban review** va chinh sua noi dung
3. **Admin panel** co tool bulk generate
4. **External links** curate tu cac nguon uy tin (MDN, W3Schools, freeCodeCamp, etc.)

### Workflow:
```
Admin tao Learning Path structure (tracks, lesson titles)
  → AI generate summary cho tung bai
  → AI generate quiz questions (3 loai)
  → Admin review va publish
```
