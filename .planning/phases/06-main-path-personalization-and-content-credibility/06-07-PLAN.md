# Plan 06-07: Ranked Recommendation Cards Frontend UI

## Goal
Refactor the onboarding recommendation frontend to display 3 ranked learning path cards with match score, profile-linked explanation, difficulty badge, and individual "Chọn lộ trình này" CTAs. Update hooks and types to match the new backend ranked array contract.

## must_haves
- useOnboardingRecommendation returns an array of 3 ranked recommendations with learningPathId
- RecommendationPanel renders 3 vertically stacked cards in rank order
- Top card has accent emphasis and "Gợi ý hàng đầu" badge
- Each card has its own "Chọn lộ trình này" CTA that submits that learningPathId
- While submitting, all 3 CTAs are disabled to prevent double submission
- Loading state shows one container-level skeleton, not 3 fake cards
- Error state uses locked copy from UI-SPEC

```typescript
export interface RankedRecommendation {
  learningPathId: string;
  pathSlug: string;
  matchScore: number;
  explanation: string;
  focusAreas: string[];
}

export interface OnboardingRecommendationResponse {
  source: 'ai' | 'fallback';
  rankings: RankedRecommendation[];
  tips: string[];
}
```

Update the hook to return the new type:
```typescript
export function useOnboardingRecommendation(enabled: boolean) {
  return useQuery({
    queryKey: ['onboarding', 'recommendation'],
    queryFn: async () => {
      const res = await api.get('/onboarding/recommendation');
      return res.data.data as OnboardingRecommendationResponse;
    },
    enabled,
  });
}
```

Remove the old `OnboardingRecommendation` interface entirely.

Rewrite `frontend/src/components/onboarding/RecommendationPanel.tsx`:

```tsx
import { Alert, AlertDescription } from '../ui/alert';
import { Badge } from '../ui/badge';
import { Button } from '../ui/button';
import { Card, CardContent } from '../ui/card';
import type { RankedRecommendation } from '../../hooks/useOnboardingRecommendation';
import { vi } from '../../strings/vi';

const PATH_NAMES: Record<string, string> = {
  'frontend-developer': 'Frontend Developer',
  'backend-developer': 'Backend Developer',
  'fullstack-developer': 'Fullstack Developer',
  'ai-python': 'AI / Data Science (Python)',
};

interface RecommendationPanelProps {
  rankings: RankedRecommendation[];
  onConfirm: (learningPathId: string) => void;
  isConfirming: boolean;
  error: string | null;
}

export default function RecommendationPanel({
  rankings,
  onConfirm,
  isConfirming,
  error,
}: RecommendationPanelProps) {
  return (
    <div className="space-y-4">
      <h2 className="text-[28px] font-semibold leading-[1.2] text-dp-text-primary">
        {vi.onboarding.recommendationSectionTitle}
      </h2>

      {error && (
        <Alert className="rounded-xl border border-red-500/30 bg-red-500/10 p-3 text-sm text-red-300">
          <AlertDescription className="text-red-300">
            {error}
          </AlertDescription>
        </Alert>
      )}

      {rankings.map((rec, index) => {
        const isTop = index === 0;
        const pathName = PATH_NAMES[rec.pathSlug] ?? rec.pathSlug;

        return (
          <Card
            key={rec.learningPathId}
            className={`rounded-2xl transition-colors ${
              isTop
                ? 'border-[#3B82F6]/40 bg-[#3B82F6]/[0.05]'
                : 'border-white/[0.08] bg-white/[0.03]'
            }`}
          >
            <CardContent className="p-6 space-y-4">
              {/* Rank + Name + Badges */}
              <div className="flex flex-wrap items-center gap-3">
                <span className="text-sm font-semibold text-dp-text-muted">
                  #{index + 1}
                </span>
                <h3 className="text-xl font-semibold text-dp-text-primary">
                  {pathName}
                </h3>
                {isTop && (
                  <Badge className="border-[#3B82F6]/30 bg-[#3B82F6]/10 text-[#3B82F6]">
                    {vi.onboarding.topRankedBadge}
                  </Badge>
                )}
              </div>

              {/* Match Score */}
              <div className="flex items-center gap-2">
                <span className="text-sm text-dp-text-muted">
                  {vi.onboarding.matchScoreLabel}:
                </span>
                <span className="font-mono text-sm font-semibold text-dp-text-primary">
                  {rec.matchScore}%
                </span>
              </div>

              {/* Explanation */}
              <p className="text-base text-dp-text-secondary leading-relaxed">
                {rec.explanation}
              </p>

              {/* Focus Areas */}
              {rec.focusAreas.length > 0 && (
                <div>
                  <p className="mb-2 text-sm font-semibold text-dp-text-muted">
                    {vi.onboarding.focusAreasLabel}
                  </p>
                  <div className="flex flex-wrap gap-2">
                    {rec.focusAreas.map((area) => (
                      <span
                        key={area}
                        className="rounded-full border border-white/[0.08] bg-white/[0.04] px-2.5 py-1 text-xs text-dp-text-secondary"
                      >
                        {area}
                      </span>
                    ))}
                  </div>
                </div>
              )}

              {/* CTA */}
              <Button
                type="button"
                onClick={() => onConfirm(rec.learningPathId)}
                disabled={isConfirming}
                className={`w-full min-h-[44px] rounded-xl font-semibold text-white transition-all disabled:opacity-40 ${
                  isTop
                    ? 'bg-[#3B82F6] hover:bg-[#3B82F6]/90 shadow-lg shadow-[#3B82F6]/20'
                    : 'bg-white/[0.08] hover:bg-white/[0.12] text-dp-text-primary'
                }`}
              >
                {isConfirming
                  ? vi.onboarding.submitting
                  : vi.onboarding.selectPathCta}
              </Button>
            </CardContent>
          </Card>
        );
      })}
    </div>
  );
}
```

The key changes:
- Props now accept `rankings: RankedRecommendation[]` instead of single recommendation
- Renders 3 cards vertically stacked
- Top card (#1) has accent border, accent badge, and filled primary CTA
- Cards #2 and #3 have neutral borders and ghost-style CTA
- All CTAs disabled when `isConfirming` is true
- Match score rendered as text percentage
