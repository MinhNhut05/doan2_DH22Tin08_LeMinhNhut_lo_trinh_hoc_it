# Plan 06-02: Ranked Recommendation Pipeline — Backend (3 Paths with Score + Explanation)

## Goal
Update the backend recommendation pipeline (prompt builder → AI → parser → fallback) to return an array of 3 ranked learning path recommendations, each with match score percentage and profile-linked explanation. Update the onboarding service to return this ranked array.

## must_haves
- AI prompt requests 3 ranked paths with matchScore (0-100) and profile-linked explanation
- Parser validates and returns an array of 3 RankedRecommendation objects
- Fallback returns 3 deterministic ranked paths with scores
- getRecommendation returns array with learningPathId resolved for each item
- Each recommendation includes: slug, matchScore, explanation, focusAreas

```typescript
export function parseRecommendation(raw: string): RecommendationResult | null {
  const cleaned = extractJsonString(raw.trim());

  let parsed: { rankings?: unknown; tips?: unknown };
  try {
    parsed = JSON.parse(cleaned) as { rankings?: unknown; tips?: unknown };
  } catch {
    return null;
  }

  // Validate rankings is an array
  if (!Array.isArray(parsed.rankings) || parsed.rankings.length === 0) {
    return null;
  }

  const validatedRankings: RankedRecommendation[] = [];
  const usedSlugs = new Set<string>();

  for (const item of parsed.rankings) {
    if (typeof item !== 'object' || item === null) continue;

    const raw = item as Record<string, unknown>;

    // Validate pathSlug
    if (typeof raw.pathSlug !== 'string') continue;
    const pathSlug = raw.pathSlug.trim().toLowerCase();
    if (!VALID_PATH_SLUGS.includes(pathSlug)) continue;
    if (usedSlugs.has(pathSlug)) continue;

    // Validate matchScore
    const matchScore = Number(raw.matchScore);
    if (isNaN(matchScore) || matchScore < 0 || matchScore > 100) continue;

    // Validate explanation
    if (typeof raw.explanation !== 'string' || raw.explanation.trim() === '') continue;

    const focusAreas = normalizeStringArray(raw.focusAreas);

    usedSlugs.add(pathSlug);
    validatedRankings.push({
      pathSlug,
      matchScore: Math.round(matchScore),
      explanation: raw.explanation.trim(),
      focusAreas,
    });
  }

  // Must have at least 1 valid ranking (ideally 3, but degrade gracefully)
  if (validatedRankings.length === 0) {
    return null;
  }

  // Sort by matchScore descending
  validatedRankings.sort((a, b) => b.matchScore - a.matchScore);

  const tips = normalizeStringArray(parsed.tips);

  return {
    source: 'ai',
    rankings: validatedRankings.slice(0, 3),
    tips,
  };
}
```

Keep `extractJsonString` and `normalizeStringArray` helpers unchanged.
