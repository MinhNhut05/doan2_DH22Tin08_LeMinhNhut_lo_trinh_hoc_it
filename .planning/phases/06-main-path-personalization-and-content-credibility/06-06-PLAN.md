# Plan 06-06: AI Chat Context Enrichment with Learner Profile and Main Path

## Goal
Refactor AiContextBuilder to inject LearnerProfile data (careerGoal, skillLevel, strengths, weaknesses, learningPace) and mainLearningPathId into the AI chat system prompt, replacing the implicit first-enrollment heuristic.

## must_haves
- AiContextBuilder reads LearnerProfile from DB for the current user
- System prompt includes learner's careerGoal, skillLevel, learningPace, strengths, weaknesses
- System prompt includes main learning path name (not just whatever enrollment came first)
- AI is instructed to tailor tone and complexity to user's skillLevel
- Fallback still works when no profile exists (generic prompt)

3. In the `buildContext` method, when `!lessonId` (no explicit lesson), replace the current `findFirst` query with a profile-aware query:

```typescript
if (!lessonId) {
  // Query learner profile to find main path (Phase 6: D-17)
  const learnerProfile = await this.prisma.learnerProfile.findUnique({
    where: { userId },
    include: {
      mainLearningPath: {
        select: { id: true },
      },
    },
  });

  // Use mainLearningPathId to find the correct UserLearningPath
  const userPath = learnerProfile?.mainLearningPathId
    ? await this.prisma.userLearningPath.findUnique({
        where: {
          userId_learningPathId: {
            userId,
            learningPathId: learnerProfile.mainLearningPathId,
          },
        },
        include: { currentLesson: true, learningPath: true },
      })
    : await this.prisma.userLearningPath.findFirst({
        where: { userId, completedAt: null },
        include: { currentLesson: true, learningPath: true },
      });

  if (userPath?.currentLessonId) {
    return this.buildContext(userId, userPath.currentLessonId);
  }

  return this.buildFallback();
}
```

4. After building the `AIContext` object (around line 148-164), add a query for the learner profile and inject its data:

```typescript
// Phase 6: Enrich context with LearnerProfile data (D-16)
const profile = await this.prisma.learnerProfile.findUnique({
  where: { userId },
  select: {
    careerGoal: true,
    skillLevel: true,
    learningPace: true,
    strengths: true,
    weaknesses: true,
  },
});

if (profile) {
  context.user.careerGoal = profile.careerGoal;
  context.user.skillLevel = profile.skillLevel;
  context.user.learningPace = profile.learningPace;

    ? (profile.strengths as string[])
    : [];
  context.user.weaknesses = Array.isArray(profile.weaknesses)
    ? (profile.weaknesses as string[])
    : [];
}
```
