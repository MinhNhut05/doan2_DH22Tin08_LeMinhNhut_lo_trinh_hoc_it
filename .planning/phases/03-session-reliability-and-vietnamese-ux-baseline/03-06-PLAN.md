# Plan 06: Fix Remaining English Labels in Dashboard and Lesson (Gap Closure)

Two critical learner-facing screens still render visible English labels that bypass `vi.ts`:
1. Dashboard: mock badge names `'First Step'`, `'7-Day Streak'`, `'Quiz Master'` are hardcoded English
2. Lesson: external link type labels `'Docs'`, `'Video'`, `'Tutorial'`, `'Course'`, `'Interactive'` are hardcoded English

This plan adds the missing Vietnamese keys to `vi.ts` and updates both pages to consume them.

## must_haves

- `vi.ts` contains Vietnamese badge name keys: `badgeFirstStepName`, `badgeSevenDayStreakName`, `badgeQuizMasterName`
- `vi.ts` contains Vietnamese resource type label keys: `resourceDocs`, `resourceVideo`, `resourceTutorial`, `resourceCourse`, `resourceInteractive`
- Dashboard.tsx `mockBadges` array uses `vi.dashboard.badgeFirstStepName` etc. instead of English strings
- Lesson.tsx `linkTypeConfig` uses `vi.lesson.resourceDocs` etc. instead of English strings
- Zero visible English labels remain in badge names or resource type labels on these two screens
